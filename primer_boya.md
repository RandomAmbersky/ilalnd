# Пример боя

Дуэль между рыцарем Альбертом и рыцарем Бернардом.

Рыцарь Альберт имеет:
* доспех ( **+1ОК** к защите )
* щит ( **1ОК** к защите, один слот )
* одноручный меч ( **1М** атака, один слот )

Рыцарь Бернард имеет:
* доспех ( **+1ОК** к защите )
* щит ( **1ОК** к защите )
* одноручный меч ( **1М** атака )
